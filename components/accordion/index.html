Accordion

Use accordions to show vertically stacked list items when clicking on an accordion header

Usage

The Accordion items wrapped in a container with the .accordion class and each accordion item consists of a header and content inside the container.

Class Description
.accordion-item Accordion item consists of header and content.
.accordion-toggle Show/Hide each accordion item content when clicking on it.
.accordion-content Contains each accordion content part
Petronia does not come up with any javascript functionality. You have to write toggle class on click yourself

To show/hide accordion content, you have to toggle class .active to .accordion-item .

Accordion Default

A standard accordion with no modifiers
